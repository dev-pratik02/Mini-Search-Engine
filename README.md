# Mini Search Engine

## Overview
This project implements a simple **Mini Search Engine** in C using **data structures** such as hash tables and linked lists. It indexes multiple text documents, supports single keyword and AND searches, and displays the top frequent words.

The system demonstrates fundamental **information retrieval concepts** using an **inverted index** and provides basic search functionality without requiring a graphical interface.

## Features
- Load and index multiple text documents
- Single keyword search with ranked results
- Boolean AND search for multiple keywords
- Display top frequent words
- View index statistics

## Requirements
- C compiler (GCC or compatible)
- Operating system: Linux, macOS, or Windows (with minor adjustments)
- Terminal/Command Line interface
